# IronStrong Fitness – Proyecto Integrador de Base de Datos

## Descripción del Proyecto
IronStrong Fitness es una cadena de gimnasios que requiere modernizar la gestión de su información mediante una base de datos centralizada.  
Este proyecto simula un entorno profesional de desarrollo, donde se implementa una base de datos relacional que administra miembros, membresías, contratos, entrenadores, clases grupales, asistencias y pagos, garantizando integridad de datos y soporte para reportes complejos.

El proyecto incluye control de versiones con Git y GitHub, programación avanzada en SQL Server y un módulo web independiente para consumo de API.

---

## Objetivo General
Diseñar e implementar una base de datos robusta para IronStrong Fitness que soporte la lógica de negocio crítica del gimnasio, asegurando integridad, trazabilidad, seguridad y generación de reportes avanzados.

---

## Tecnologías Utilizadas
- SQL Server Express
- Git y GitHub
- HTML5
- Bootstrap
- jQuery
- API pública TheMealDB
